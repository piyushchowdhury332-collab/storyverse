# 📚 StoryVerse - AI Story Generator

## 🚀 Overview
StoryVerse is an AI-powered story generator that uses Google's Gemini API to create creative stories based on user input.

## ✨ Features
- AI story generation using Gemini
- Genre, tone, and length selection
- Story history
- Word count & reading time
- Download story as text file
- Clean modern UI

## 🛠 Tech Stack
- Python
- Streamlit
- Google Gemini API
